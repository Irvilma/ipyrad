The ipyrad Ethos
================

* Simplicity: Code should be easy to install and use.
* Resourcefulness: Example tutorials are available for all types of assemblies.
* Reproducibility: We promote the use of Jupyter Notebooks to organize workflows.
* Flexibility: API access to functions and data to build custom assemblies.
* Transparency: Providing human readable code and data files.
